Map Info

This choropleth map visualizes annual county level unemployment rates across the United States from 2001 through 2021. The map allows users to explore spatial and temporal changes in unemployment patterns by moving the year slider.

County boundaries are joined to unemployment data using county FIPS codes. A sequential color scheme highlights relative unemployment levels while state boundaries are drawn above counties to improve geographic interpretation.

This interactive design enables users to compare spatial patterns of unemployment across different years and observe how those patterns shift over time.
